## ***Landing Page- Путешествия по России***

Проект создан в рамках учебного курса Web-разработчик и был направлен на изучение возможностей адаптивной вёрстки (320px-1280px) На странице рассказывается о путешествиях по России. Пользователь может посмотреть фотографии, прочитать об интересных фактах, пройдя по ссылкам - получить дополнительную информацию при планировании путешествия.
